Using Goldmine to help visualize data

Accelerate your ability to extract valuable data from lists of objects. Think of it as Ruby's Enumerable#group_by on steroids.

The example below uses New York wifi hotspot data from NYC Open Data to illustrate just how simple it can be to extract meaning from a dataset.

Checkout the project to learn how this page was created with less than 250 lines of code.

  • Data
  • Ruby
  • HTML

Of course we are leverageing several libraries to make life easy.

  • JQuery
  • Bootstrap
  • Google Chart

Note: This example project manages all concerns in just 2 files. A formal project would lend itself to a more elegant solution.
